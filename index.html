Welcome to clusterbot’s documentation!¶

Introduction¶

ClusterBot is a Python application written to create XMPP based services without the need to program and provides means to manage an ssh server. It builds on JabberBot, which in turn builds on xmpppy. The License used is `GPLv3 <http://www.gnu.org/licenses/>.

Features¶

  • Add commands via XML files
  • Add events via XML files
  • Events are polled automatically and the user is informed if they appear
  • Subscribe/Unsubscribe from events
