getConstructorParameterAssignments()¶
Assumes the existence of a local dictionary with the name args. :returns: A statement which takes each key in parameters as a local variable and assigns the corresponding value of the key with the same name found in args. If there is no key with the same name in args, the statement assigns to it the corresponding value of parameters as a default value.
getConstructorParameters(startWithComma=False, withDefaultValues=False)¶
Parameters:
  • startWithComma – If True, start the returned string with a comma iff the returned string contains one or more parameters
  • withDefaultValues – If True, every parameter in the returned string is substituted by paramM=defaultValueM, where defaultValueM is the corresponding value found in the dictionary parameters storing the parameter’s names.
Returns:

A string of the form ‘param1,param2...paramN’, where param1 through paramN are all keys of parameters
getInstanceVariableAssignments()¶
Returns:A string of the form ‘self.param1 = defaultValue1;nself.param2 = defaultValue2;n..self.paramN = defaultValueN;n’, where param1 through paramN are all keys of parameters and defaultValue1 through defaultValueN are the corresponding values.
