The ClusterBot Module¶

class clusterbot.ClusterBot.ClusterBot(jabberID, jabberPW, allowedJIDs, headnodeIP, headnodeUser, headnodePW, debug=False)¶

Bases: jabberbot.JabberBot

A Jabber service to monitor clusters.

  • Has a standing ssh connection to the headnode of a cluster to send commands via the ssh module.
  • Allows for whitelisting by Jabber ids.
  • Supports runtime loading of new command modules (see importnewcommands()).
